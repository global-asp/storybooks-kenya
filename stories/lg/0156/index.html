{% include picturecol.html index="02" %} {% include textcol_start.html %} Waliwo Ggonya nga muyala nnyo. {% include textcol_end.html %}

Once there was a very hungry crocodile.

Wakati mmoja palikuwa na mamba mwenye njaa.

{% include audiocontrols.html index="02" prev="01" next="03" %}
{% include picturecol.html index="03" %} {% include textcol_start.html %} Buli kiseera yasooba nga mpolampola okunonya ekyokulya. Noluvanyuma..... {% include textcol_end.html %}

He searched for food slowly and quietly. And then...

Alitafuta chakula pole pole tena kimya. Halafu...

{% include audiocontrols.html index="03" prev="02" next="04" %}
{% include picturecol.html index="04" %} {% include textcol_start.html %} POO!!! Ggonya nayasama!! {% include textcol_end.html %}

POW!!! The crocodile strikes!

POO!!! Mamba anashambulia!

{% include audiocontrols.html index="04" prev="03" next="05" %}
{% include picturecol.html index="05" %} {% include textcol_start.html %} Oluvanyuma lw'ekyo Ggonya yali akuse era nga musanyufu. {% include textcol_end.html %}

After that he is no longer hungry, and he is happy.

Baada ya hiyo hana njaa tena,na anafurahi.

{% include audiocontrols.html index="05" prev="04" next="06" %}
{% include picturecol.html index="06" %} {% include textcol_start.html %} Paka lwadamu okuba omuyala. {% include textcol_end.html %}

Until he gets hungry again.

Hadi anapohisi njaa tena.
