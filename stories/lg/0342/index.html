{% include picturecol.html index="02" %} {% include textcol_start.html %} Olunaku lumu maama yale ebibala, ennanansi, amenvu, emicu emiyembe amapeera. Yabiwanika ku kabada wag. {% include textcol_end.html %}

One day, mama got a lot of fruit.

Siku moja mamangu alinunua matunda mengi sana.

{% include audiocontrols.html index="02" prev="01" next="03" %}
{% include picturecol.html index="03" %} {% include textcol_start.html %} Twabuuza maama nti "Ffe tunaalya ddi ku bibala?" Maama yaddamu nti, "Tujja kulya olwaleero." {% include textcol_end.html %}

"When can we have some fruit?" we ask. "We will eat the fruit tonight," says mama.

Mimi na ndugu zangu tulijiuliza lini mama angetugawia ili tule.

{% include audiocontrols.html index="03" prev="02" next="04" %}
{% include picturecol.html index="04" %} {% include textcol_start.html %} Muganda wange Kayima wamululu nnyo. Yagenda alumaluma ku buli kibala. ebisinga yeyabirya. Bweyamala okubirya ye nagenda okuzannya. {% include textcol_end.html %}

My brother Rahim is greedy. He tastes all the fruit. He eats a lot of it.

Baadaye, kakangu, Rahim, alijificha. Aliyala matunda yale yote peke yake!

{% include audiocontrols.html index="04" prev="03" next="05" %}
{% include picturecol.html index="05" %} {% include textcol_start.html %} Muto wange omulala yagamba nti "Mazima labayo kayima kyakoze!" Kayima si muntu mulungi ate alina nnyo omululu. {% include textcol_end.html %}

"Look at what Rahim did!" shouts my little brother. "Rahim is naughty and selfish," I say.

Mimi na ndugu yangu tulipogundua, tulilia tukisema, “Rahim ni mtukutu tena ni mchoyo.”

{% include audiocontrols.html index="05" prev="04" next="06" %}
{% include picturecol.html index="06" %} {% include textcol_start.html %} Maama yanyigira nnyo Kayima. {% include textcol_end.html %}

Mother is angry with Rahim.

Mama alikasirika sana alipogundua kwamba matunda yote yalikuwa yameliwa.

{% include audiocontrols.html index="06" prev="05" next="07" %}
{% include picturecol.html index="07" %} {% include textcol_start.html %} Era naffe twanyigira nnyo Kayima. Kyokka yye Kayima nga teyefiirayo. {% include textcol_end.html %}

We are also angry with Rahim. But Rahim is not sorry.

Mimi na ndugu yangu tulimwangalia Rahim kwa macho makali.

{% include audiocontrols.html index="07" prev="06" next="08" %}
{% include picturecol.html index="08" %} {% include textcol_start.html %} Muto wange yabuuza maama nti "Kayima togenda kumubonereza?" {% include textcol_end.html %}

"Aren't you going to punish Rahim?" asks little brother.

Tulmwuliza mama ikiwa angemwadhibu Rahim.

{% include audiocontrols.html index="08" prev="07" next="09" %}
{% include picturecol.html index="09" %} {% include textcol_start.html %} "Kayima, olina okutwentondera mangu ddala" {% include textcol_end.html %}

"Rahim, soon you will be sorry," warns mama.

Lakini, Rahim alipata adhabu ambayo hatukutarajia.

{% include audiocontrols.html index="09" prev="08" next="10" %}
{% include picturecol.html index="10" %} {% include textcol_start.html %} Kati kayima tawulira bulungi. {% include textcol_end.html %}

Rahim starts to feel sick.

Rahim hakuweza kutulia pahali pamoja!

{% include audiocontrols.html index="10" prev="09" next="11" %}
{% include picturecol.html index="11" %} {% include textcol_start.html %} Kayima yayogera mu kaama nti "Olubuto lunnuma nnyo." {% include textcol_end.html %}

"My tummy is so sore," whispers Rahim.

Mama alipomwuuliza alijibu kwa sauti ya chini, “Naumwa na tumbo.”

{% include audiocontrols.html index="11" prev="10" next="12" %}
{% include picturecol.html index="12" %} {% include textcol_start.html %} Maama yali yakitegedde dda nti olubuto lujja kumuluma. Kati ebibala bibonereza Kayima! {% include textcol_end.html %}

Mama knew this would happen. The fruit is punishing Rahim!

Sote tuliona kwamba hiyo ilikuwa adhabu mbaya kuliko kama Rahim angechapwa na mama.

{% include audiocontrols.html index="12" prev="11" next="13" %}
{% include picturecol.html index="13" %} {% include textcol_start.html %} Oluvanyuma Kayima yatusaba tumusonyiwe. Yagamba nti "sikyaddamu kubeera na mululu mulundi mulala" naffe twamusonyiwa. {% include textcol_end.html %}

Later, Rahim says sorry to us. "I will never be so greedy again," he promises. And we all believe him.

Baadaye alipopata nafuu, Rahim alituomba msamaha.
