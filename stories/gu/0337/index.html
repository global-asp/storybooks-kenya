{% include picturecol.html index="02" %} {% include textcol_start.html %} એક દિવસ, એક સુખી કુટુંબ હતી. {% include textcol_end.html %}

Once upon a time, there lived a happy family.

Hapo zamani za kale, paliishi familia yenye furaha.

{% include audiocontrols.html index="02" prev="01" next="03" %}
{% include picturecol.html index="03" %} {% include textcol_start.html %} તેઓ એ કોઈ દિવસ ઝગડા ના કર્યા. તેઓ તેના મા બાપ ને ઘર માં અને ખેત માં મદદ કર્યા. {% include textcol_end.html %}

They never fought with each other. They helped their parents at home and in the fields.

Hawakugombana hata kidogo. Watoto waliwasaidia wazazi nyumbani na shambani.

{% include audiocontrols.html index="03" prev="02" next="04" %}
{% include picturecol.html index="04" %} {% include textcol_start.html %} પણ આગ પાસે જવા ની રજા નતી. {% include textcol_end.html %}

But they were not allowed to go near a fire.

Lakini, walikatazwa kwenda karibu na moto.

{% include audiocontrols.html index="04" prev="03" next="05" %}
{% include picturecol.html index="05" %} {% include textcol_start.html %} બધું કામ રત્ન કરવા નું હતું, કારણ કે તેઓ મીણ થઈ બાળકો હતા! {% include textcol_end.html %}

They had to do all their work during the night. Because they were made of wax!

Walifanya kazi zao zote usiku. Hii ni kwa sababu walikuwa wameumbwa kwa nta.

{% include audiocontrols.html index="05" prev="04" next="06" %}
{% include picturecol.html index="06" %} {% include textcol_start.html %} પણ એક દિવસ, એક છોકરો ને બહારે તડકા માં જવું તું. {% include textcol_end.html %}

But one of the boys longed to go out in the sunlight.

Lakini, kijana mmoja alitamani kwenda nje wakati wa mchana.

{% include audiocontrols.html index="06" prev="05" next="07" %}
{% include picturecol.html index="07" %} {% include textcol_start.html %} એક દિવસ તેને બહુ જવું તું. એના ભાઈઉ એને ચેતવણી આપી... {% include textcol_end.html %}

One day the longing was too strong. His brothers warned him...

Siku moja tamaa ya kwenda nje ikazidi. Wenzake wakamwonya asiende juani. Lakini, hakusikia.

{% include audiocontrols.html index="07" prev="06" next="08" %}
{% include picturecol.html index="08" %} {% include textcol_start.html %} પણ બહુ મોડું થયું! એ તાપ માં ઓગળી ગયો. {% include textcol_end.html %}

But it was too late! He melted in the hot sun.

Aliyeyuka kwa sababu joto lilikuwa kali.

{% include audiocontrols.html index="08" prev="07" next="09" %}
{% include picturecol.html index="09" %} {% include textcol_start.html %} એને જોય ને બીજા મીણ ના બાળકો ઉદાસ હતા. {% include textcol_end.html %}

The wax children were so sad to see their brother melting away.

Watoto wa nta walihuzunika walipoona kaka yao akiyeyuka.

{% include audiocontrols.html index="09" prev="08" next="10" %}
{% include picturecol.html index="10" %} {% include textcol_start.html %} પણ તેઓ પાસે એક યોજના હતી. તેઓ મીણ માં થી એક પક્ષી બનાવુ. {% include textcol_end.html %}

But they made a plan. They shaped the lump of melted wax into a bird.

Wakapanga jambo fulani. Walichukua nta iliyoyeyuka wakatengeza ndege.

{% include audiocontrols.html index="10" prev="09" next="11" %}
{% include picturecol.html index="11" %} {% include textcol_start.html %} પક્ષી ભાઈ લય ને એક ઊંચો પહાડ પર લય ગયા. {% include textcol_end.html %}

They took their bird brother up to a high mountain.

Wakamweka ndege huyo juu ya mlima mrefu.

{% include audiocontrols.html index="11" prev="10" next="12" %}
{% include picturecol.html index="12" %} {% include textcol_start.html %} જ્યારે સૂરજ ઉગવા માંડ્યું, એ ગીત ગાયું અને ઉડી ગયો. {% include textcol_end.html %}

And as the sun rose, he flew away singing into the morning light.

Jua lilipochomoza asubuhi, akapeperuka akiimba kwa furaha.
