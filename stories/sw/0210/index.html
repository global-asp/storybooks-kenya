Alizoea kuchunga ng'ombe na bibi yake.

One day the soldiers came.

Siku moja wanajeshi walikuja.

They took the cows away.

Wakawachukua ng'ombe.

{% include picturecol.html index="06" %} {% include textcol_start.html %} Tingi na bibi yake walikimbia na kujificha. {% include textcol_end.html %}

Tingi and his grandmother ran away and hid.

Tingi na bibi yake walikimbia na kujificha.

{% include audiocontrols.html index="06" prev="05" next="07" %}
{% include picturecol.html index="07" %} {% include textcol_start.html %} Walijificha vichakani hadi usiku. {% include textcol_end.html %}

They hid in the bush until night.

Walijificha vichakani hadi usiku.

{% include audiocontrols.html index="07" prev="06" next="08" %}
{% include picturecol.html index="08" %} {% include textcol_start.html %} Wanajeshi walirudi tena. {% include textcol_end.html %}

Then the soldiers came back.

Wanajeshi walirudi tena.

{% include audiocontrols.html index="08" prev="07" next="09" %}
{% include picturecol.html index="09" %} {% include textcol_start.html %} Bibi alimficha Tingi chini ya matawi ya mti. {% include textcol_end.html %}

Grandmother hid Tingi under the leaves.

Bibi alimficha Tingi chini ya matawi ya mti.

{% include audiocontrols.html index="09" prev="08" next="10" %}

One of the soldiers put his foot right on him, but he kept quiet.

Mwanajeshi mmoja alimkanyaga, lakini akanyamaza kimya.

{% include audiocontrols.html index="10" prev="09" next="11" %}

When it was safe, Tingi and his grandmother came out.

Wanajeshi walipo ondoka Tingi na bibi yake wakajitokeza.

{% include audiocontrols.html index="11" prev="10" next="12" %}
{% include picturecol.html index="12" %} {% include textcol_start.html %} Wakajikongoja kimya kimya kurudi nyumbani. {% include textcol_end.html %}

They crept home very quietly.

Wakajikongoja kimya kimya kurudi nyumbani.
