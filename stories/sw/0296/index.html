{% include picturecol.html index="02" %} {% include textcol_start.html %} Tom anabeba tenga la ndizi zilizo iva. {% include textcol_end.html %}

Tom carries a tray of ripe bananas.

Tom anabeba tenga la ndizi zilizo iva.

{% include audiocontrols.html index="02" prev="01" next="03" %}
{% include picturecol.html index="03" %} {% include textcol_start.html %} Tom anenda sokoni kuuza ndizi. {% include textcol_end.html %}

Tom goes to the market to sell bananas.

Tom anenda sokoni kuuza ndizi.

{% include audiocontrols.html index="03" prev="02" next="04" %}
{% include picturecol.html index="04" %} {% include textcol_start.html %} Sokoni watu wana nunua matunda. {% include textcol_end.html %}

People at the market are buying fruit.

Sokoni watu wana nunua matunda.

{% include audiocontrols.html index="04" prev="03" next="05" %}
{% include picturecol.html index="05" %} {% include textcol_start.html %} Hakuna anaye nunua ndizi za Tom. Wanapenda kununua ndizi kutoka kwa akina mama. {% include textcol_end.html %}

But no one is buying Tom’s bananas. They prefer to buy fruit from women.

Hakuna anaye nunua ndizi za Tom. Wanapenda kununua ndizi kutoka kwa akina mama.

{% include audiocontrols.html index="05" prev="04" next="06" %}
{% include picturecol.html index="06" %} {% include textcol_start.html %} "Kwenye jamii yetu, ni wanawake pekee ndio wanaouza matunda," watu wanasema. "Ni mwanaume wa aina gani huyu?" watu wanahoji. {% include textcol_end.html %}

"In our community, only women sell fruit," people say. "What kind of a man is this?" people ask.

"Kwenye jamii yetu, ni wanawake pekee ndio wanaouza matunda," watu wanasema. "Ni mwanaume wa aina gani huyu?" watu wanahoji.

{% include audiocontrols.html index="06" prev="05" next="07" %}
{% include picturecol.html index="07" %} {% include textcol_start.html %} Lakini Tom hakati tamaa. Anaita, "Nunua ndizi zangu! Nunueni ndizi zangu tamu!" {% include textcol_end.html %}

But Tom does not give up. He calls, “Buy my bananas! Buy my sweet ripe bananas!”

Lakini Tom hakati tamaa. Anaita, "Nunua ndizi zangu! Nunueni ndizi zangu tamu!"

{% include audiocontrols.html index="07" prev="06" next="08" %}
{% include picturecol.html index="08" %} {% include textcol_start.html %} Mwanamke mmoja anachukua kichana cha ndizi kutoka kwenye tenga. Anaziangalia ndizi kwa umakini. {% include textcol_end.html %}

One woman picks up a bunch of bananas from the tray. She looks at the bananas carefully.

Mwanamke mmoja anachukua kichana cha ndizi kutoka kwenye tenga. Anaziangalia ndizi kwa umakini.

{% include audiocontrols.html index="08" prev="07" next="09" %}
{% include picturecol.html index="09" %} {% include textcol_start.html %} Mwanamke ananunua ndizi. {% include textcol_end.html %}

The woman buys the bananas.

Mwanamke ananunua ndizi.

{% include audiocontrols.html index="09" prev="08" next="10" %}
{% include picturecol.html index="10" %} {% include textcol_start.html %} Watu wengi wanakuja kwenye kichanja. Wananunua ndizi za Tom na kuzila. {% include textcol_end.html %}

More people come to the stall. They buy Tom’s bananas and eat them.

Watu wengi wanakuja kwenye kichanja. Wananunua ndizi za Tom na kuzila.

{% include audiocontrols.html index="10" prev="09" next="11" %}
{% include picturecol.html index="11" %} {% include textcol_start.html %} Mara, tenga linakuwa tupu. Tom anahesabu pesa alizopata. {% include textcol_end.html %}

Soon, the tray is empty. Tom counts the money he earned.

Mara, tenga linakuwa tupu. Tom anahesabu pesa alizopata.

{% include audiocontrols.html index="11" prev="10" next="12" %}
{% include picturecol.html index="12" %} {% include textcol_start.html %} Tom ananunua sabauni, sukari, na mkate. Anaweka vitu kwenye tenga lake. {% include textcol_end.html %}

Then Tom buys soap, sugar, and bread. He puts the things in his tray.

Tom ananunua sabauni, sukari, na mkate. Anaweka vitu kwenye tenga lake.

{% include audiocontrols.html index="12" prev="11" next="13" %}
{% include picturecol.html index="13" %} {% include textcol_start.html %} Tom anajitwisha tenga kichwani na kwenda nyumbani. {% include textcol_end.html %}

Tom balances the tray on his head and goes home.

Tom anajitwisha tenga kichwani na kwenda nyumbani.
