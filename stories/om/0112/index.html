Qaama kiyya My body Mwili wangu

Clare Verbeek, Thembani Dladla, Zanele Buthelezi

Mlungisi Dlamini

Alemu Ifa, Gelgelo Belambel, Godana Jirmo

Oromo

Level 1

The audio for this story is currently not available.

Fiiguun danda’aan.

I can run with it.

Ninaweza kukimbia nao.

Kuruuphachuun danda’aan.

I can jump with it.

Ninaweza kuruka nao.

Shubbisuun danda’aan.

I can dance with it.

Ninaweza kucheza nao.

Bishaan daakuun danda’aan.

I can swim with it.

Ninaweza kuogelea nao.

Bira dabruun danda’aan.

I can skip with it.

Ninaweza kuruka kamba nao.

Dhiituun danda’aan.

I can kick with it.

Ninaweza kupiga mpira nao.

Miliquun danda’aan.

I can escape with it.

Ninaweza kutoroka nao.

Garuu, gonkumaa barrisuu hin danda’uun.

But I can never fly with it.

Lakini siwezi kupaa nao angani.

Written by: Clare Verbeek, Thembani Dladla, Zanele Buthelezi
Illustrated by: Mlungisi Dlamini
Translated by: Alemu Ifa, Gelgelo Belambel, Godana Jirmo
Language: Oromo
Level: Level 1
Source: My body from African Storybook
