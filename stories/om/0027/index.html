My village had many problems. We made a long line to fetch water from one tap.

Kijiji changu kilikuwa na matatizo mengi. Tulipanga foleni ndefu tukisubiri kuchota maji kutoka mfereji mmoja.

{% include audiocontrols.html index="02" prev="01" next="03" %}
{% include picturecol.html index="03" %} {% include textcol_start.html %} Nyaata harjooman dhufee eggachu qabna. {% include textcol_end.html %}

We waited for food donated by others.

Tulisubiri chini ya mti mkubwa kupokea msaada wa chakula.

{% include audiocontrols.html index="03" prev="02" next="04" %}
{% include picturecol.html index="04" %} {% include textcol_start.html %} Sabaabaa hattu tif yeeroo dhan mana chufachu qabna. {% include textcol_end.html %}

We locked our houses early because of thieves.

Tulifunga nyumba zetu mapema kwa ajili ya wizi.

{% include audiocontrols.html index="04" prev="03" next="05" %}
{% include picturecol.html index="05" %} {% include textcol_start.html %} Ijooleen bayeen mana barumsaa kessaa harca'uu. {% include textcol_end.html %}

Many children dropped out of school.

Baadhi ya watoto waliacha shule kwa sababu wazazi wao hawakuweza kulipa karo.

{% include audiocontrols.html index="05" prev="04" next="06" %}
{% include picturecol.html index="06" %} {% include textcol_start.html %} Shaamareen mana namaa biraati aka hojatuti qaxaramanii ganda biraati hojatu. {% include textcol_end.html %}

Young girls worked as maids in other villages.

Wasichana wadogo walifanya kazi kijijini kama yaya.

{% include audiocontrols.html index="06" prev="05" next="07" %}
{% include picturecol.html index="07" %} {% include textcol_start.html %} Dargagonni immo nannoo ganda olgadi demo ; kaaneen namaf qotu. {% include textcol_end.html %}

Young boys roamed around the village while others worked on people's farms.

Wavulana wadogo waliranda randa kijijini huku wengine wakifanyia watu kazi mashambani.

{% include audiocontrols.html index="07" prev="06" next="08" %}
{% include picturecol.html index="08" %} {% include textcol_start.html %} Yeroo bubben ka'u warqaa fi balfi garagaraa dallaafi mukken irrati tuluma. {% include textcol_end.html %}

When the wind blew, waste paper hung on trees and fences.

Upepo ulipovuma, ulipeperusha karatasi chafu na kuziangika mitini na uani.

{% include audiocontrols.html index="08" prev="07" next="09" %}
{% include picturecol.html index="09" %} {% include textcol_start.html %} Namooni bayeen biriciqoon muramu. {% include textcol_end.html %}

People were cut by broken glass that was thrown carelessly.

Njiani, watu walikatwa na vipande vya chupa vilivyotupwa ovyo ovyo.

{% include audiocontrols.html index="09" prev="08" next="10" %}
{% include picturecol.html index="10" %} {% include textcol_start.html %} Gaftokko bishhan ujummo dha cabee bishaan dhabamee ture. {% include textcol_end.html %}

Then one day, the tap dried up and our containers were empty.

Halafu siku moja, mfereji wa maji ulikauka na mitungi yetu ikawa tupu.

{% include audiocontrols.html index="10" prev="09" next="11" %}
{% include picturecol.html index="11" %} {% include textcol_start.html %} Abbon ko mana irra gara mana adeeme ummata walgahiig wamee ture. {% include textcol_end.html %}

My father walked from house to house asking people to attend a village meeting.

Babangu alitembea kutoka nyumba hadi nyumba akiwaalika watu kuhudhuria mkutano wa kujiji.

{% include audiocontrols.html index="11" prev="10" next="12" %}
{% include picturecol.html index="12" %} {% include textcol_start.html %} Nammoni muka jalti walgahanii and waldhagefatan. {% include textcol_end.html %}

People gathered under a big tree and listened.

Watu walikusanyika chini ya mti ule ule ambako tulipokea chakula cha msaada.

{% include audiocontrols.html index="12" prev="11" next="13" %}
{% include picturecol.html index="13" %} {% include textcol_start.html %} Abbon ko olka'ee akan jedhe, "Rakko keynaa hikuuf waliwajjiin hajachuu qabna." {% include textcol_end.html %}

My father stood up and said, “We need to work together to solve our problems.”

Babangu alisimama katikati ya kikundi na kusema, “Lazima turekebishe kijiji chetu.”

{% include audiocontrols.html index="13" prev="12" next="14" %}
{% include picturecol.html index="14" %} {% include textcol_start.html %} Mucaan ganna sadeeti, Jumaan damee mukaa gubba ta'ee akam jedhe, "Ani nanno kenya ququleesun nagummacha jedhe." {% include textcol_end.html %}

Eight-year-old Juma, sitting on a tree trunk shouted, “I can help with cleaning up.”

Juma, aliyekuwa amepanda mti, alisema kwa sauti, “Naweza kusafisha uwanja wa nyumba yetu.”

{% include audiocontrols.html index="14" prev="13" next="15" %}
{% include picturecol.html index="15" %} {% include textcol_start.html %} Dubartiin tokko akan jete, "Duburatonni kun nawajjiin gamtofeene midhaan facaasuu nidandenyaa." {% include textcol_end.html %}

One woman said, “The women can join me to grow food.”

Mwanamke mmoja wa makamo, alisema, “Sisi wanawake tunaweza kuungana na kupanda chakula.”

{% include audiocontrols.html index="15" prev="14" next="16" %}
{% include picturecol.html index="16" %} {% include textcol_start.html %} "Dubartiin biraa olkatee akan jete," namooni dhiira immo bolla yaqotan. {% include textcol_end.html %}

Another man stood up and said, “The men will dig a well.”

Mwanamme mwingine alisimama na kusema, “Wanaume watachimba kisima cha maji.”

{% include audiocontrols.html index="16" prev="15" next="17" %}
{% include picturecol.html index="17" %} {% include textcol_start.html %} Hundi kenyaa sagalee tokkon iyyinee akan jenne, "Hardha irra jalqabnee waliwajjiin hijachuun rakko keyna hikinee". {% include textcol_end.html %}

We all shouted with one voice, “We must change our lives.” From that day we worked together to solve our problems.

Sote tulisema kwa sauti moja, “Lazima tubadilishe maisha yetu.” Tangu siku hiyo, tulifanya kazi pamoja tukatatua shida zetu.
