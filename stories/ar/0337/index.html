{% include picturecol.html index="02" %} {% include textcol_start.html %} فِي يَوْمٍ مِنَ الأَيَّامِ، كَانَتْ هُنَاكَ عَائِلَةٌ سَعِيدَةٌ. {% include textcol_end.html %}

Once upon a time, there lived a happy family.

Hapo zamani za kale, paliishi familia yenye furaha.

{% include audiocontrols.html index="02" prev="01" next="03" %}
{% include picturecol.html index="03" %} {% include textcol_start.html %} لَمْ يَحْدُثْ بَيْنَهُمْ أَيُّ خِلَافٍ قَطُّ. كَانُوا دَائِمًا يُعَاوِنُونَ وَالِدَيْهِمْ فِي الحَقْلِ وَفِي المَنْزِلِ. {% include textcol_end.html %}

They never fought with each other. They helped their parents at home and in the fields.

Hawakugombana hata kidogo. Watoto waliwasaidia wazazi nyumbani na shambani.

{% include audiocontrols.html index="03" prev="02" next="04" %}
{% include picturecol.html index="04" %} {% include textcol_start.html %} لَكِنْ لَمْ يُسْمَحْ لَهُمْ بِالاِقْتِرَابِ مِنَ النَّارِ أَبَداً. {% include textcol_end.html %}

But they were not allowed to go near a fire.

Lakini, walikatazwa kwenda karibu na moto.

{% include audiocontrols.html index="04" prev="03" next="05" %}
{% include picturecol.html index="05" %} {% include textcol_start.html %} كَانَ عَلَيْهِمْ القِيَامُ بِكُلِّ مَا لَدَيْهِمْ مِنْ أَعْمَالٍ خِلَالَ اللَّيلِ، لِأَنَّهُمْ مَصْنُوعُونَ مِنَ الشَّمْعِ. {% include textcol_end.html %}

They had to do all their work during the night. Because they were made of wax!

Walifanya kazi zao zote usiku. Hii ni kwa sababu walikuwa wameumbwa kwa nta.

{% include audiocontrols.html index="05" prev="04" next="06" %}
{% include picturecol.html index="06" %} {% include textcol_start.html %} وَلَكِنَّ أَحَدَ الأَوْلَادِ كَانَ يَتُوقُ إِلَى الخُرُوجِ فِي ضَوْءِ الشَّمْسِ. {% include textcol_end.html %}

But one of the boys longed to go out in the sunlight.

Lakini, kijana mmoja alitamani kwenda nje wakati wa mchana.

{% include audiocontrols.html index="06" prev="05" next="07" %}
{% include picturecol.html index="07" %} {% include textcol_start.html %} وَذَاتَ يَوْمٍ اشْتَدَّتْ رَغْبَتُهُ فِي الخُرُوجِ فِي النَّهَارِ، فَحَذَّرَهُ إِخْوَتُهُ… {% include textcol_end.html %}

One day the longing was too strong. His brothers warned him...

Siku moja tamaa ya kwenda nje ikazidi. Wenzake wakamwonya asiende juani. Lakini, hakusikia.

{% include audiocontrols.html index="07" prev="06" next="08" %}
{% include picturecol.html index="08" %} {% include textcol_start.html %} وَلَكِنْ بَعْدَ فَوَاتِ الأَوَانِ! فَقَدْ ذَابَ فِي الشَّمْسِ الحَارِقَةِ. {% include textcol_end.html %}

But it was too late! He melted in the hot sun.

Aliyeyuka kwa sababu joto lilikuwa kali.

{% include audiocontrols.html index="08" prev="07" next="09" %}
{% include picturecol.html index="09" %} {% include textcol_start.html %} كَانَ أَطْفَالُ الشَّمْسِ حَزِينِينَ لِلْغَايَةِ لِرُؤْيَةِ أَخِيهِمْ يَذُوبُ. {% include textcol_end.html %}

The wax children were so sad to see their brother melting away.

Watoto wa nta walihuzunika walipoona kaka yao akiyeyuka.

{% include audiocontrols.html index="09" prev="08" next="10" %}
{% include picturecol.html index="10" %} {% include textcol_start.html %} إِلَّا أَنَّهُمْ قَامُوا بِتَشْكِيلِ قِطْعَةِ الشَّمْعِ المُنْصَهِرِ إِلَى طَيْرٍ. {% include textcol_end.html %}

But they made a plan. They shaped the lump of melted wax into a bird.

Wakapanga jambo fulani. Walichukua nta iliyoyeyuka wakatengeza ndege.

{% include audiocontrols.html index="10" prev="09" next="11" %}
{% include picturecol.html index="11" %} {% include textcol_start.html %} بَعْدَ ذَلِكَ، أَخَذُوا أَخَاهُمْ المُتَشَكِّلَ عَلَى هَيْئَةِ طَيْرٍ إِلَى جَبَلٍ عَالٍ. {% include textcol_end.html %}

They took their bird brother up to a high mountain.

Wakamweka ndege huyo juu ya mlima mrefu.

{% include audiocontrols.html index="11" prev="10" next="12" %}
{% include picturecol.html index="12" %} {% include textcol_start.html %} وَمَعَ بُزُوغِ الشَّمْسِ، طَارَ بَعِيداً وَهْوَ يُغَنِّي فِي ضَوْءِ النَّهَارِ. {% include textcol_end.html %}

And as the sun rose, he flew away singing into the morning light.

Jua lilipochomoza asubuhi, akapeperuka akiimba kwa furaha.
