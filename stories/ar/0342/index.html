{% include picturecol.html index="02" %} {% include textcol_start.html %} فِي أحَدِ اَلأيَّامِ حَصَلَتْ مَامَا عَلَى اَلكَثِيرِ مِنَ الفَواكِهَ. {% include textcol_end.html %}

One day, mama got a lot of fruit.

Siku moja mamangu alinunua matunda mengi sana.

{% include audiocontrols.html index="02" prev="01" next="03" %}
{% include picturecol.html index="03" %} {% include textcol_start.html %} سَأَلْنَاهَا: "مَتى يُمْكِنُنا أّن نَأخُذَ القَليلَ مِنَها؟" أَجَابَتْ: "سَنَأكُلُ الفَوَاكِهَ في المَسَاءِ". {% include textcol_end.html %}

"When can we have some fruit?" we ask. "We will eat the fruit tonight," says mama.

Mimi na ndugu zangu tulijiuliza lini mama angetugawia ili tule.

{% include audiocontrols.html index="03" prev="02" next="04" %}
{% include picturecol.html index="04" %} {% include textcol_start.html %} أَخِي رَهِيمْ كَانَ طَمَّاعًا، تَذَوَّقَ كُلَّ الفَواكِهِ، وَأَكَلَ الكَثِيرَ مِنْهَا. {% include textcol_end.html %}

My brother Rahim is greedy. He tastes all the fruit. He eats a lot of it.

Baadaye, kakangu, Rahim, alijificha. Aliyala matunda yale yote peke yake!

{% include audiocontrols.html index="04" prev="03" next="05" %}
{% include picturecol.html index="05" %} {% include textcol_start.html %} صَرَخَ أَخي الصَّغِيرُ: "انْظُرْ مَاذَا فَعَلَ رَهِيمْ"! أَمَّا أَنَا فَقُلتُ: "رَهِيمْ غَيْرُ مُحْتَرَمٍ وَأَنَانِيٌّ". {% include textcol_end.html %}

"Look at what Rahim did!" shouts my little brother. "Rahim is naughty and selfish," I say.

Mimi na ndugu yangu tulipogundua, tulilia tukisema, “Rahim ni mtukutu tena ni mchoyo.”

{% include audiocontrols.html index="05" prev="04" next="06" %}
{% include picturecol.html index="06" %} {% include textcol_start.html %} غَضِبَتْ مَامَا مِنْ رَهِيمْ. {% include textcol_end.html %}

Mother is angry with Rahim.

Mama alikasirika sana alipogundua kwamba matunda yote yalikuwa yameliwa.

{% include audiocontrols.html index="06" prev="05" next="07" %}
{% include picturecol.html index="07" %} {% include textcol_start.html %} وَنَحْنُ أَيْضًا غَضِبْنَا مِنْ رَهِيمْ ولَكِنَّهُ لَمْ يَعْتَذِرْ. {% include textcol_end.html %}

We are also angry with Rahim. But Rahim is not sorry.

Mimi na ndugu yangu tulimwangalia Rahim kwa macho makali.

{% include audiocontrols.html index="07" prev="06" next="08" %}
{% include picturecol.html index="08" %} {% include textcol_start.html %} سَأَلَ أَخِي الصَّغِيرُ: "أَلَنْ تُعَاقِبِي رَهِيمْ؟" {% include textcol_end.html %}

"Aren't you going to punish Rahim?" asks little brother.

Tulmwuliza mama ikiwa angemwadhibu Rahim.

{% include audiocontrols.html index="08" prev="07" next="09" %}
{% include picturecol.html index="09" %} {% include textcol_start.html %} حَذَّرَتْ مَامَا رَهِيمْ فَقَاَلتْ: "عَلَيْكَ أَنْ تَعْتَذِرَ حَالَّا". {% include textcol_end.html %}

"Rahim, soon you will be sorry," warns mama.

Lakini, Rahim alipata adhabu ambayo hatukutarajia.

{% include audiocontrols.html index="09" prev="08" next="10" %}
{% include picturecol.html index="10" %} {% include textcol_start.html %} بَدَأَ رَهِيمْ بِالشُّعُورِ بِالمَرَضِ. {% include textcol_end.html %}

Rahim starts to feel sick.

Rahim hakuweza kutulia pahali pamoja!

{% include audiocontrols.html index="10" prev="09" next="11" %}
{% include picturecol.html index="11" %} {% include textcol_start.html %} هَمَسَ رَهِيمْ: "مَعِدَتِي تُؤْلِمُنِي". {% include textcol_end.html %}

"My tummy is so sore," whispers Rahim.

Mama alipomwuuliza alijibu kwa sauti ya chini, “Naumwa na tumbo.”

{% include audiocontrols.html index="11" prev="10" next="12" %}
{% include picturecol.html index="12" %} {% include textcol_start.html %} عَرَفَتْ مَامَا مَاذَا حَصَلَ: "إنَّ الفَوَاكِهَ عَاقَبَتْ رَهِيمْ"! {% include textcol_end.html %}

Mama knew this would happen. The fruit is punishing Rahim!

Sote tuliona kwamba hiyo ilikuwa adhabu mbaya kuliko kama Rahim angechapwa na mama.

{% include audiocontrols.html index="12" prev="11" next="13" %}
{% include picturecol.html index="13" %} {% include textcol_start.html %} أَخِيرًا اِعْتَذَرَ رَهِيمْ فَقَالَ لَنَا: "لَنْ أَكُونَ طَمَّاعاً بَعْدَ اليَوْمِ". {% include textcol_end.html %}

Later, Rahim says sorry to us. "I will never be so greedy again," he promises. And we all believe him.

Baadaye alipopata nafuu, Rahim alituomba msamaha.
