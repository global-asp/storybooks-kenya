His hoe was too short.

Jembe lake lilikuwa fupi mno.

{% include audiocontrols.html index="02" prev="01" next="03" %}
{% include picturecol.html index="03" %} {% include textcol_start.html %} وَمَدْخَلُ مَنْزِلِهِ مُنْخَفضاً جِدًّا. {% include textcol_end.html %}

His doorway was too low.

Mlango wake ulikuwa chini mno.

{% include audiocontrols.html index="03" prev="02" next="04" %}
{% include picturecol.html index="04" %} {% include textcol_start.html %} كَانَ فِرَاشُهُ قَصِيرًا جِدًّا. {% include textcol_end.html %}

His bed was too short.

Kitanda chake kilikuwa kifupi mno.

{% include audiocontrols.html index="04" prev="03" next="05" %}
{% include picturecol.html index="05" %} {% include textcol_start.html %} وَكَذَلِكَ دَرَّاجَتَهُ الهَوَائِيَّةُ. {% include textcol_end.html %}

His bicycle was too short.

Baisikeli yake ilikuwa fupi mno.

{% include audiocontrols.html index="05" prev="04" next="06" %}
{% include picturecol.html index="06" %} {% include textcol_start.html %} إِنَّهُ لَرَجُلٌ طَويلٌ جِدًّا. {% include textcol_end.html %}

This man was too tall!

Mtu huyu alikuwa mrefu zaidi.

{% include audiocontrols.html index="06" prev="05" next="07" %}
{% include picturecol.html index="07" %} {% include textcol_start.html %} وَلِذَلِكَ فَقَدْ قامَ بِصِناعَةِ مَسْكَةٍ طَويلَةٍ لِمِجْرَفَتِهِ. {% include textcol_end.html %}

He made a very long hoe handle.

Alitengeneza mpini wa jembe mrefu sana.

{% include audiocontrols.html index="07" prev="06" next="08" %}
{% include picturecol.html index="08" %} {% include textcol_start.html %} وَوَسَّعَ بابَ مَنْزِلِهِ. {% include textcol_end.html %}

He made very high door frames.

Alitengeneza fremu refu za mlango.

{% include audiocontrols.html index="08" prev="07" next="09" %}
{% include picturecol.html index="09" %} {% include textcol_start.html %} كَما قَامَ بِصِناعَةِ فِرَاشٍ طَوِيلٍ جِدًّا. {% include textcol_end.html %}

He made a very long bed.

Alitengeneza kitanda kirefu sana.

{% include audiocontrols.html index="09" prev="08" next="10" %}
{% include picturecol.html index="10" %} {% include textcol_start.html %} وَاشْتَرَى دَرَّاجَةً هَوَائِيَّةً عَالِيَةً. {% include textcol_end.html %}

He bought a very high bicycle.

Kisha akanunua baiskikeli iliyo refu mno.

{% include audiocontrols.html index="10" prev="09" next="11" %}
{% include picturecol.html index="11" %} {% include textcol_start.html %} لَقَدْ جَلَسَ عَلَى كُرْسِيٍّ مُرْتَفِعٍ وتَناوَلَ طَعامَهُ بِشَوْكَةٍ طَوِيلَةٍ. {% include textcol_end.html %}

He sat on a very high chair. He ate with a very long fork.

Aliketi juu ya kiti kilichokuwa juu sana. Alikula kwa kutumia uma iliyokuwa refu mno.

{% include audiocontrols.html index="11" prev="10" next="12" %}
{% include picturecol.html index="12" %} {% include textcol_start.html %} تَرَكَ الرَّجُلُ الطَّويلُ مَنْزِلَهُ وَذَهَبَ لِلْعَيْشِ فِي غَابَةٍ كَبِيرَةٍ حَيْثُ أَمْضَى سِنِيناً عَديدَةً مِنْ عُمْرِهِ هُنَاكَ. {% include textcol_end.html %}

He left his house and lived in a big forest. He lived for many years.

Aliacha nyumba yake na kuishi katika msitu mkubwa. Aliishi kwa miaka mingi.
