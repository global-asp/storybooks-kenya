{% include picturecol.html index="02" %} {% include textcol_start.html %} المَاعِزُ والكَلْبُ والبَقَرَةُ كَانُوا أَصْدِقاءَ جِدًّا. في أَحَدِ الأَيَّام ذَهَبُوا فِي رِحْلَةٍ بِسَيَّارةِ أُجْرَةٍ. {% include textcol_end.html %}

Goat, Dog and Cow were great friends. One day they went on a journey in a taxi.

Mbuzi, Mbwa na Ng’ombe walikuwa marafiki wakubwa. Siku moja walienda safari kwa teksi.

{% include audiocontrols.html index="02" prev="01" next="03" %}
{% include picturecol.html index="03" %} {% include textcol_start.html %} عِنْدَمَا وَصَلُوا إِلَى نِهايَةِ الرِّحْلَةِ، طَلَبَ مِنْهُمْ السائِقُ أُجْرَةَ رِحْلَتِهِمْ. دَفَعَتْ البَقَرَةُ مَا عَلَيْهَا. {% include textcol_end.html %}

When they reached the end of their journey, the driver asked them to pay their fares. Cow paid her fare.

Walipofika mwisho wa safari yao, dereva aliwauliza walipe nauli zao. Ng’ombe alilipa nauli yake.

{% include audiocontrols.html index="03" prev="02" next="04" %}
{% include picturecol.html index="04" %} {% include textcol_start.html %} دَفَعَ الكَلْبُ زِيَادَةً، لأِنَّهُ لَمْ يَكُنْ يَعْرِفُ المَبْلَغَ المَطْلُوبَ. {% include textcol_end.html %}

Dog paid a bit extra, because he did not have the correct money.

Mbwo alilipa zaidi kidogo kwa sababu hakuwa na kiasi kamili cha pesa.

{% include audiocontrols.html index="04" prev="03" next="05" %}
{% include picturecol.html index="05" %} {% include textcol_start.html %} بَيْنَمَا كَانَ السَّائِقُ يُعْطِي الكَلْبَ بَاقِيَ النُقودَ، هَرَبَتْ المَاعِزُ بَعيداً. {% include textcol_end.html %}

The driver was about to give Dog his change when Goat ran away without paying anything.

Dereva alikuwa karibu kumrejeshea Mbwa pesa zake zilizosalia wakati Mbuzi alipotoroka bila kulipa chochote.

{% include audiocontrols.html index="05" prev="04" next="06" %}
{% include picturecol.html index="06" %} {% include textcol_start.html %} غَضِبَ السَّائِقُ جِدّاً فَتَرَكَهُمْ وَرَحَلَ بِسَيَّارَتِهِ دُونَ أَنْ يُعْطِيَ الكَلْبَ بَاقِيَ نُقُودِهِ. {% include textcol_end.html %}

The driver was very annoyed. He drove away without giving Dog his change.

Dereva alikasirika sana. Aliondoka huku akiendesha gari bila kumrudishia Mbwa pesa zake zilizobaki.

{% include audiocontrols.html index="06" prev="05" next="07" %}
{% include picturecol.html index="07" %} {% include textcol_start.html %} لِهَذا السَّبَبِ يَرْكُضُ الكَلْبُ حَتَّى اليَوْمِ، أَمَامَ كُلِّ سيَّارَةٍ، وَيَنْظُرُ إِلَيْهَا نَظْرَةً سَريعةً. لِلْبَحْثِ عَنْ السَّائِقِ الذِي أَخَذَ بَاقِيَ النُّقُودِ. {% include textcol_end.html %}

That is why, even today, Dog runs towards a car to peep inside and find the driver who owes him his change.

Hiyo ndiyo sababu, hata leo Mbwa hukimbia kuelekea gari akiwa na nia ya kuangalia ndani ili kumpata dereva aliye na pesa zake zilizosalia.

{% include audiocontrols.html index="07" prev="06" next="08" %}
{% include picturecol.html index="08" %} {% include textcol_start.html %} أَمَّا المَاعِزُ فَقَدْ كانَتْ تَهْرُبُ عِنْدَمَا تَسْمَعُ صَوْتَ أَيِّ سَيَّارَةٍ. فَهْيَ تَخَافُ أَنْ يُقْبَضَ عَلَيْهَا لِأَنَّهَا لَمْ تَدْفَعْ مَا عَلَيْهَا مِنَ النُقودِ. {% include textcol_end.html %}

Goat runs away from the sound of a car. She is afraid she will be arrested for not paying her fare.

Mbuzi naye hutoroka anaposikia mlio wa gari. Anaogopa kushikwa kwa kukosa kulipa nauli yake.

{% include audiocontrols.html index="08" prev="07" next="09" %}
{% include picturecol.html index="09" %} {% include textcol_start.html %} أَمَّا البَقَرَةُ فَهْيَ لَا تَهْتَمُّ عِنْدَمَا تَأْتِي أَيُّ سَيَّارَةٍ. وَتَأْخُذُ كُلَّ وَقْتِهَا بِعُبُورِ الشَّارِعَ لِأَنَّها تَعْرِفُ أَنَّهَا دَفَعَتْ كُلَّ مَا عَلَيْهَا كَامِلاً. {% include textcol_end.html %}

And Cow is not bothered when a car is coming. Cow takes her time crossing the road because she knows she paid her fare in full.

Na Ng’ombe hashughuliki gari linapo kuja. Ng’ombe huchukuwa muda wake kupita barabara kutoka upande mmoja hadi mwingine. Anajua kuwa alilipa nauli yake kamili!
