{% include picturecol.html index="02" %} {% include textcol_start.html %} يَنامُ أَخِي الصَّغِيرُ مُتَأَخِّرًا جِدًّا. أَنَا اَسْتيْقِظُ بَاكِراً لِأَنَّنِي مُمَيَّزَةٌ. {% include textcol_end.html %}

My little brother sleeps very late. I wake early, because I am great!

Mdogo wangu anachelewa kulala. Naamka mapema kwasababu mimi ni mkubwa

{% include audiocontrols.html index="02" prev="01" next="03" %}
{% include picturecol.html index="03" %} {% include textcol_start.html %} أَنَا وَاحِدَةٌ مِنَ اَلَّذينَ يَسْمَحُونَ لِلشَّمْسِ أَن تَدْخُلَ. {% include textcol_end.html %}

I am the one who lets in the sun.

Mimi ndiye ninaye ruhusu jua kuingia ndani.

{% include audiocontrols.html index="03" prev="02" next="04" %}
{% include picturecol.html index="04" %} {% include textcol_start.html %} مَامَا تقُولُ لِي: "أَنْتِ نَجْمَتِي اَلصَّباحيّةُ". {% include textcol_end.html %}

“You're my morning star,” says Ma.

"Wewe ni nyota yangu ya asubuhi," anasema mama.

{% include audiocontrols.html index="04" prev="03" next="05" %}
{% include picturecol.html index="05" %} {% include textcol_start.html %} أَنَا اَسْتَحِمُّ وَحْدِي، أَنَا لَا أَحْتَاجُ لِمُسَاعَدَةِ أَحَدٍ. {% include textcol_end.html %}

I wash myself, I don't need any help.

Naoga mwenyewe, sihitaji msaada.

{% include audiocontrols.html index="05" prev="04" next="06" %}
{% include picturecol.html index="06" %} {% include textcol_start.html %} أَنَا يُمْكِنُنِي أَنْ أَتَعَامَلَ مَعَ المَاءِ البَارِدِ وَالرَّائِحَةُ اَلكَرِيهَةُ لِلصَّابُونَةِ الزَّرْقَاءِ. {% include textcol_end.html %}

I can cope with cold water and blue smelly soap.

Naweza kuvumilia maji ya baridi na harufu nzuri ya sabuni ya bluu.

{% include audiocontrols.html index="06" prev="05" next="07" %}
{% include picturecol.html index="07" %} {% include textcol_start.html %} مَامَا تُذَكِّرُنِي: "لَا تَنْسَيْ أَسْنَانَكِ"، فَأُجِيبُهَا: "أَنَا مُسْتَحِيلْ!" {% include textcol_end.html %}

Ma reminds, "Don't forget teeth." I reply, "Never, not me!"

Mama ananikumbusha, "Usihashau kupiga mswaki." Namjibu, "Kamwe, labda sio mim!"

{% include audiocontrols.html index="07" prev="06" next="08" %}
{% include picturecol.html index="08" %} {% include textcol_start.html %} بَعْدَ اَلاِسْتِحْمَامِ، أُسَلِّمُ عَلَى جَدِّي وَعَمَّتِي، وَأُوَدِّعُهُمْ. {% include textcol_end.html %}

After washing, I greet Grandpa and Auntie, and wish them a good day.

Baada ya kuoga , nawasalimia babu na shangazi, na kuwatakia siku njema.

{% include audiocontrols.html index="08" prev="07" next="09" %}
{% include picturecol.html index="09" %} {% include textcol_start.html %} ثُمَّ أَلْبِسُ ثِيَابِي بِنَفْسِي، وَأَقُولُ: "أَنَا الآنَ كَبِيرَةٌ". {% include textcol_end.html %}

Then I dress myself, “I'm big now Ma,” I say.

Najivisha nguo mwenyewe, "Mimi ni mkubwa sasa mama," Nina mwambia.

{% include audiocontrols.html index="09" prev="08" next="10" %}
{% include picturecol.html index="10" %} {% include textcol_start.html %} أَنَا اَسْتَطِيعُ أَنْ أُغْلِقَ أَزْرَارِي وَأَرْبِطَ حِذَائِي. {% include textcol_end.html %}

I can close my buttons and buckle my shoes.

Ninaweza kufunga vishikizo na kurekebisha viatu vyangu.

{% include audiocontrols.html index="10" prev="09" next="11" %}
{% include picturecol.html index="11" %} {% include textcol_start.html %} أَنَا أَتَأَكَّدُ مِنْ أَنَّ أَخِي اَلصَّغِيرُ يَعْرفُ كُلَّ أَخْبَارِ اَلمَدْرَسَةِ. {% include textcol_end.html %}

And I make sure little brother knows all the school news.

Na ninahakikisha mdogo wangu anapata habari zote za shule.

{% include audiocontrols.html index="11" prev="10" next="12" %}
{% include picturecol.html index="12" %} {% include textcol_start.html %} أَبْذُلُ جُهْدِي فِي الصَّفِّ لِأَقُومَ بِالأفْضَلِ بِكُلِّ شَيْءٍ. {% include textcol_end.html %}

In class I do my best in every way.

Darasani najitahidi sana.

{% include audiocontrols.html index="12" prev="11" next="13" %}
{% include picturecol.html index="13" %} {% include textcol_start.html %} أَنَا أَقُومُ بِكُلِّ هَذِهِ اَلأفْعَالِ كُلَّ يَوْمٍ. لَكِنَّ أَكْثَرَ الأَشْيَاءِ التِي أُحِبُّهَا اللَّعِبُ. ثُمَّ اللَّعِبُ. {% include textcol_end.html %}

I do all these good things every day. But the thing I like most, is to play and play!

Ninafanya hivi vitu vyote vizuri kila siku, ila kitu ninacho kipenda sana ni kucheza na kucheza.
