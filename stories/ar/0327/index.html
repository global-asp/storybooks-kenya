Two giraffes are going to drink water.

Twiga wawili wanaenda kunywa maji.

{% include audiocontrols.html index="03" prev="02" next="04" %}
{% include picturecol.html index="04" %} {% include textcol_start.html %} ثَلاَثَة جَوَامِيسَ وَأَرْبَعَةُ طُيُورٍ ذَاهِبُونَ أَيْضًا لِشُرْبِ المَاءِ. {% include textcol_end.html %}

Three buffaloes and four birds are also going to drink water.

Nyati watatu na ndege wanne nao wanaenda kunywa maji.

{% include audiocontrols.html index="04" prev="03" next="05" %}
{% include picturecol.html index="05" %} {% include textcol_start.html %} خَمْسَةُ ظِبَاءٍ وَسِتَّةُ خَنَازِيرَ بَرِّيَّةٍ يَسِيرُونَ نَحْوَ المَاءِ. {% include textcol_end.html %}

Five impalas and six wild pigs are walking to the water.

Swala watano na ngiri sita wanatembea kuelekea kwenye maji.

{% include audiocontrols.html index="05" prev="04" next="06" %}
{% include picturecol.html index="06" %} {% include textcol_start.html %} سَبْعَةُ أَحْمِرَةٍ وَحْشِيَّةٍ تَجْرِي نَحْوَ المَاءِ. {% include textcol_end.html %}

Seven zebras are running to the water.

Punda milia saba wanakimbia kuelekea kwenye maji.

{% include audiocontrols.html index="06" prev="05" next="07" %}
{% include picturecol.html index="07" %} {% include textcol_start.html %} ثَمَانِي ضَفَادِعَ وَتِسْعُ سَمَكَاتٍ يَسْبَحْنَ فِي المَاءِ. {% include textcol_end.html %}

Eight frogs and nine fish are swimming in the water.

Vyura wanane na samaki tisa wanaogelea kwenye maji.

{% include audiocontrols.html index="07" prev="06" next="08" %}

One lion roars. He also wants to drink. Who is afraid of the lion?

Simba mmoja ananguruma. Naye anataka kunywa maji. Nani anamwogopa simba?

{% include audiocontrols.html index="08" prev="07" next="09" %}
{% include picturecol.html index="09" %} {% include textcol_start.html %} فِيلٌ وَاحِدٌ يَشْرَبُ المَاءَ مَعَ الأَسَدِ. {% include textcol_end.html %}

One elephant is drinking water with the lion.

Tembo mmoja anakunywa maji na simba.
