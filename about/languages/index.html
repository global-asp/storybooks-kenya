Languages

The following languages are currently included in the scope of the Storybooks Kenya project:

  • English
  • Swahili
  • Amharic
  • Arabic
  • Embu / Kîembu
  • Gujarati
  • Ekegusii
  • Kamba / Kikamba
  • Gikuyu / Kikuyu
  • Kalenjin
  • Luganda
  • Olukhayo
  • Luo / Dholuo
  • Luhya / Oluluhya
  • Pokomo / Kipfokomo
  • Maa / Maasai
  • Meru / Kimîîru
  • Giryama / Kigiryama
  • Oromo / Afaan Oromo
  • Rendille
  • Samburu
  • Somali
  • Turkana / Ng‘aturkana

Please note that some translations and recordings are not yet complete.

We are always looking for translators! If you speak one of the above languages and would like to help us translate or record stories, please contact us at globalafricanstorybook [at] gmail [dot] com
